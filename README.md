# 🏎️ Comic-Con India: Multi-Zone Event Parking System

A robust database design for managing high-volume parking logistics during large-scale events like Comic-Con. This system handles various vehicle categories (EVs, SUVs, Bikes), specialized reservation zones (VIP, Cosplayers, Exhibitors), and real-time session tracking.

## 📌 Database Schema Overview

This system is designed to solve the complexities of event-based parking where:
* Vehicles enter and exit multiple times over a multi-day event.
* Specific spots are reserved for stakeholders (Staff, VIPs, Creators).
* Payments are calculated dynamically based on vehicle type and duration.
* Real-time availability must be tracked across different levels and zones.

## 🏗️ Data Architecture

The system is modeled into three core layers: **Infrastructure**, **Entities**, and **Transactions**.
